Prototyping kit

This prototype kit provides a simple way to make interactive prototypes. These prototypes can be used to show ideas to people you work with, and to do user reasearch.

When using any of the styles or designing content for your prototype remember to refer to the design manual.

Getting started

You will need a text editor to work with the prototype kit. We recommend the Atom text editor. It's free and works on both Mac and Windows.

Introducing Atom

Atom has a useful ‘Get to know Atom’ section when you first open it up. It is recommended you read these steps to get you set up and familiar working in the Atom environment. There are some parts of the setup that might be beyond your level, it’s fine to skip through these if you don’t feel comfortable with them yet. It will not effect your workflow on this project. However, it is recommended that you complete the ‘Open your project’ tab.

Open your project in Atom

  1. Open Atom.
  2. Within the right hand tab of Atom, under ‘Get to know Atom!’, click on the first tab ‘Open a project’.
  3. Clicking this opens an accordion with the button ‘Open a project’, this button opens up a finder navigator.
  4. Navigate to the root of your new prototyping project (the ‘root’ is the file that holds all your project files). Click once on the root file to highlight it and click open. For example, if your root file is in your 'Documents' folder you will navigate to this file on your system, find your prototyping file and click it once to highlight it and click 'Open'.
  5. This then opens the folder in a new, left-hand pane.

Adding pages

To get started on your prototype:

  1. Copy and rename one of the templates to index.html - this will be your homepage.
  2. Use the code examples in this page to add content to your template.
  3. When you need another page, reapeat this process - naming the new page my-other-page.html or whatever you like.

Linking pages

Pages are linked together with anchor tags. To link index.html to my-other-page.html use <a href="my-other-page.html">Link text</a> within the content of index.html.

Linking with buttons

Pages can be linked with buttons. This might be you can simulate a user completing a series of forms. To do this use: <a href="my-other-page.html" class="btn btn-primary">Button text</a>.

Templates

There is a one column template and a two column template to get you started with some common page layouts. If you want to create your own layouts study the grid page of the design manual.
